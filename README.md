## GitHub Profile Finder

A GitHub Profile Finder built using HTML, CSS, and JavaScript.

## 🚀 Live Demo

https://wanshikarajput.github.io/Github-profile-finder/

## 📂 Repository

https://github.com/WanshikaRajput/Github-profile-finder/tree/main

## ✨ Features

- Search GitHub users
- View profile information
- Loading state
- User not found handling
- Display latest 5 repositories
- Clickable repository links
- GitHub Battle Mode
- Compare two GitHub users based on total repository stars

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- GitHub REST API
- Fetch API
- Async/Await
- Promise.all()
